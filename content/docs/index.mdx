The **Agent Shell Protocol** (ASP) is a neutral, ownable identity primitive
for AI agents. These docs are the canonical technical reference for everyone
who ships, runs, or studies the protocol.

## Pick your door

- [I ship services](/docs/builders) — register a service, drop in the
  TypeScript SDK, choose your verifier.
- [I run an agent](/docs/operators) — mint a shell, manage signers, recover
  from key loss, transfer ownership.
- [I study the protocol](/docs/protocol) — contracts, EIP-712 signatures,
  replay-resistance, trust layers, economic design.

## What you get

- **One identity, every service.** A single shell carries identity across
  every registered service on the network.
- **Replay-safe by construction.** EIP-712 + key epoch + service nonce —
  every signed action is unique, signed, and auditable.
- **Open, neutral, transparent.** All code MIT, all standards open, all
  decisions on-chain.
